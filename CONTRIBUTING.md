# Contributing to GalleryPack

## Bug reports and feature requests

GitHub Issues are welcome for:
- Bug reports (with reproduction steps)
- Feature requests and suggestions
- Questions about usage or architecture

Please open an issue at https://github.com/pvollenweider/gallerypack/issues

## Code contributions

**External code contributions are not accepted at this time without explicit prior written agreement.**

This restriction exists to preserve GalleryPack's dual-licensing model (AGPL-3.0 + commercial). Any code contributed to this repository must be licensable under both the AGPL and a commercial license. Without a Contributor License Agreement (CLA), that is not possible.

This is not a statement about the quality of external contributions — it is a legal and commercial necessity.

If you are interested in contributing code, please open an issue first to discuss. A formal CLA process may be established in the future.

## Security vulnerabilities

Please do not open public issues for security vulnerabilities. Contact the maintainer directly via GitHub.
